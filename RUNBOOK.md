# Runbook

## Démarrage
- run.bat / run.sh
- accès : http://IP:8501

## Arrêt
- Ctrl+C dans le terminal

## Logs
- logs Streamlit standard (console)

## Problèmes fréquents
- Port 8501 bloqué → ouvrir firewall
- Mauvais CSV → vérifier en-têtes
- Mot de passe oublié → reset démo
